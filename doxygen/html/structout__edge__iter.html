Detailed Description

template<typename BaseIter, typename NodeDescriptor, typename EdgeDescriptor, typename Difference>
struct out_edge_iter< BaseIter, NodeDescriptor, EdgeDescriptor, Difference >

The iterator iterates through every outgoing edge from one node in a graph.

Template Parameters
BaseIter- The type of the iterator containing the edges.
NodeDescriptor
EdgeDescriptor
Difference- The type to calculate the difference between two iterators
